# Drizzle (×2 super-resolution stacking) — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add an optional **Drizzle ×2** integration mode to the stacker — recovering sub-pixel detail from dithered Seestar subs — behind a single data-aware "Drizzle — extra detail (×2)" checkbox, with sigma-clip cleanliness preserved and a suitability gate that tells the user whether it will help.

**Architecture:** Drizzle is a new integration method that reuses the existing registration (astroalign 3×3 transforms) and frame grading. We feed the STScI `drizzle` library a per-frame `pixmap` (built from that transform ×2) and a per-frame weight mask (from a 2×-grid sigma-clip pass), producing a 2× master. A pure suitability function drives a warn-but-allow checkbox in the Stack dialog.

**Tech Stack:** Python 3.13, NumPy, the `drizzle` library (2.2.0, BSD-3, numpy-only), PySide6 (Stack dialog), pytest + pytest-qt.

```python
from __future__ import annotations

import numpy as np

DRIZZLE_SCALE = 2      # ×2 output grid — the only supported factor (kept simple)
KERNEL = "square"      # flux-conserving (gaussian/lanczos do NOT conserve flux)
PIXFRAC = 0.6          # drop shrink factor; tuned on real data in 0.5–0.7


def build_pixmap(matrix: np.ndarray, in_shape: tuple[int, int], scale: int) -> np.ndarray:
    """Per-input-pixel output (x, y) coords for drizzle. `matrix` maps input (x,y)
    to reference (x,y); the drizzle output grid is the reference scaled by `scale`."""
    h, w = in_shape
    ys, xs = np.mgrid[0:h, 0:w]
    pts = np.stack([xs.ravel(), ys.ravel(), np.ones(h * w)], axis=0)   # (3, N) = (x,y,1)
    out = np.asarray(matrix, dtype=np.float64) @ pts
    xo = out[0] / out[2] * scale
    yo = out[1] / out[2] * scale
    return np.stack([xo.reshape(h, w), yo.reshape(h, w)], axis=-1).astype(np.float64)
```

```python
def drizzle_integrate(items, in_shape, n_channels, *, pixfrac=PIXFRAC):
    from drizzle.resample import Drizzle
    h, w = in_shape
    out_shape = (h * DRIZZLE_SCALE, w * DRIZZLE_SCALE)
    drizzlers = [Drizzle(kernel=KERNEL, out_shape=out_shape) for _ in range(n_channels)]
    for data, matrix, weight in items:
        pixmap = build_pixmap(matrix, in_shape, DRIZZLE_SCALE)
        data = np.asarray(data, np.float32)
        for c in range(n_channels):
            chan = data[..., c] if data.ndim == 3 else data
            drizzlers[c].add_image(chan, exptime=1.0, pixmap=pixmap,
                                   weight_map=weight, pixfrac=pixfrac, in_units="counts")
    wht = np.maximum(drizzlers[0].out_wht, 1e-8)
    chans = [(d.out_img / wht) for d in drizzlers]        # weight-normalized average
    return np.stack(chans, axis=-1).astype(np.float32)
```
(Note: `in_units="counts"` accumulates counts and weight separately; dividing `out_img` by `out_wht` gives the weighted mean. The constant-field test pins that this normalization is right — if it fails, adjust `in_units`/normalization until a constant input reproduces.)
