# Nocturne Website Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** A self-contained static landing page for `nocturne.stehn.com` (showcase + download), deployable by dropping `site/` into an nginx web root.

**Architecture:** One `index.html` (semantic sections) + one `styles.css` (custom-property design system, responsive) + a tiny `main.js` (smooth-scroll + footer year). No framework, no build step, no external CDN/font/network calls. Content derived from the repo `README.md`.

**Tech Stack:** HTML5, modern CSS (custom properties, grid/flex, `clamp()`), vanilla JS; Python/pytest only for a static link-check test.

## Global Constraints

- Files live under `site/`. No external network requests at runtime (no CDN fonts/scripts/analytics) — system font stack only.
- Palette (CSS custom properties): `--bg:#070c1a`, `--surface:#0e1730`, `--text:#e7ecf5`, `--muted:#9aa6bd`, `--accent:#2dd4bf`, `--accent-2:#3f79d8`.

- Must include: the "not affiliated with ZWO / GraXpert / RC-Astro" line, the unsigned-beta install caveat, the GraXpert-required / RC-Astro-optional note, a GPLv3 footer.
- Responsive (feature grid 3→2→1; nav stacks on mobile); respect `prefers-reduced-motion`.

- [ ] **Step 1: Write the failing test**

Create `tests/test_site_links.py`:

```python
import re
from pathlib import Path

SITE = Path(__file__).resolve().parent.parent / "site"
HTML = (SITE / "index.html").read_text(encoding="utf-8")


def test_local_assets_exist():
    # every local src/href (not http, not #, not mailto) must exist on disk
    refs = re.findall(r'(?:src|href)="([^"]+)"', HTML)
    local = [r for r in refs if not r.startswith(("http", "#", "mailto:", "//"))]
    assert local, "expected some local asset references"
    for r in local:
        assert (SITE / r).exists(), f"missing local asset: {r}"


def test_required_content_present():
    assert "https://nocturne.stehn.com/download/Nocturne.zip" in HTML
    assert "GPLv3" in HTML or "GNU General Public License" in HTML
    assert "not affiliated" in HTML.lower()
    assert "graxpert" in HTML.lower() and "rc-astro" in HTML.lower()


def test_no_external_resource_calls():
    # no CDN/font/script includes — privacy + offline-safe
    assert "cdn" not in HTML.lower()
    assert "fonts.googleapis" not in HTML.lower()
    assert "<script" in HTML  # local main.js only
```
